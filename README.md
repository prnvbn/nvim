# nvim

This contains my neovim config. To use this config, run the following commands:

```bash
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}

# clone the repo
git clone https://github.com/prnvbn/nvim  ~/.config/nvim
```
